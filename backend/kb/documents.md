# Documents

## What this covers

Uploading, organizing, and managing the knowledge documents that back your compliance runs.

## Steps to upload a document

1. Go to `/projects` and open the project you want to add documents to.
2. Click **Upload Document**.
3. Select a PDF, DOCX, or XLSX file (max 10 MB).
4. The document is processed and indexed automatically -- this usually takes a few seconds.

## Supported formats

- PDF (`.pdf`) -- full text extraction
- Word documents (`.docx`)
- Excel spreadsheets (`.xlsx`, `.xlsm`)

## Organizing documents

- Documents belong to a **project**. They are not shared across projects automatically.
- You can upload multiple documents per project.
- Delete a document by opening it in the document list and using the remove action.

## Common errors

- **"Invalid PDF file signature"** -- the file may be corrupted or renamed. Try re-exporting it.
- **"File too large"** -- split the file or compress images before uploading.
- **"Storage limit reached"** -- your plan's storage quota is full. Go to `/settings/billing` to upgrade.
